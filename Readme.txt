Projet : Lifap 5 Programmation web

----------------------
Projet de gestionnaire de photo, qui permette de gérer une collection de photo, chaque photo est
associé à un ensemble de tags qui permettent de constituer des albums.
----------------------


Liste des fichiers : 
---------------------------------
	LIFAP5-Projet.html
	LIFAP5-Projet.js
	LIFAP5-Projet-promises.js
	Readme.txt

Exécution :
-----------

	Il faut lancer le fichier LIFAP5-Projet.html pour lancer le gestionnaire de photo
les différentes fonctionnalitées sont : 
-L'affichage des photos de l'utilisateur dans une galerie, la galerie peut changer de taille.
-L'affichage d'une photo particulière qui est sélectionnée dans la galerie.
-Filtrage des photos qui ont un certain tag, avec un menu permettant de sélectionner un tag dans la liste de tous les tags utilisés dans la collection.
-On peut modifier la description d'une photo et les tags qui sont associés aux photos.
-On peut ajouter une photo à la collection de l'utilisateur, en précisant sa description et ses tags.
-On peut modifier le nombre de photos affichées sur chaque ligne de la galerie.
-On peu filtrer les photos par date.
-Systeme de cgestion des utilisateurs via leur clé.
